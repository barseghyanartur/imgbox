Features
========
- Draw boxes over images.

Prerequisites
=============
Python 3.8+

Installation
============
pip
---

.. code-block:: sh

    pip install snapbox

Documentation
=============
- Documentation is available on `Read the Docs`_.
- For guidelines on contributing check the `Contributor guidelines`_.

Usage
=====
.. code-block:: sh

    snapbox

Tests
=====

Run the tests with unittest:

.. code-block:: sh

    python -m unittest snapbox.py

Or pytest:

.. code-block:: sh

    pytest
